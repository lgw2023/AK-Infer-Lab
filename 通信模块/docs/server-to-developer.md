# 服务器 → 开发人员邮件模板

服务器使用 `../send_notify.py` 向开发人员发送邮件。默认收件人同时使用 `gwlee1995@gmail.com,yilili1023@gmail.com`。下面模板可作为主题和正文参考。

## 任务完成

主题：`[AK服务器] 任务完成：<任务名>`

正文：

```text
任务：
主机：
时间：
结果摘要：
输出路径：
后续建议：
```

## 运行失败

主题：`[AK服务器] 运行失败：<任务名>`

正文：

```text
任务：
主机：
时间：
失败阶段：
错误摘要：
日志路径：
需要开发机补充的信息：
```

## 资源告警

主题：`[AK服务器] 资源告警：<资源类型>`

正文：

```text
主机：
时间：
资源类型：CPU/NPU/内存/磁盘/网络
当前状态：
影响范围：
建议处理：
```

## 通信配置变更或自检

主题：`[AK服务器] 通信配置更新：<范围>`

正文：

```text
主机：
时间：
变更范围：SMTP / proxychains / shell HTTP proxy / 收件人
是否涉及密钥轮换：是/否
脱敏配置摘要：
验证命令：
验证结果：
需要开发机同步的文档或代码：
```

注意：不要在邮件正文或附件中发送 `.env`、SMTP 授权码、代理账号、代理密码、服务器账号、私钥、Cookie 或其他敏感信息。需要共享密钥时，只说明“服务器本地 .env 已更新”或“proxychains 本机配置已更新”。
