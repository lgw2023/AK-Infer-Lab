# 服务器 → 开发人员邮件模板

服务器使用 `../send_notify.py` 向开发人员发送邮件。默认收件人同时使用 `gwlee1995@gmail.com,yilili1023@gmail.com`。下面模板可作为主题和正文参考。

## 任务完成

主题：`[AK服务器] 任务完成：<任务名>`

正文：

```text
任务：
主机：
时间：
结果摘要：
输出路径：
后续建议：
```

## 运行失败

主题：`[AK服务器] 运行失败：<任务名>`

正文：

```text
任务：
主机：
时间：
失败阶段：
错误摘要：
日志路径：
需要开发机补充的信息：
```
