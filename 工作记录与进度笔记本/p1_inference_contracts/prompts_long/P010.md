# P010 Long Context MoE Pressure

This is a static 32K target-bucket fixture. The manifest carries the manual token estimate; this file intentionally avoids a tokenizer dependency.

Context bundle outline:

1. Long system instructions about evidence-bound server profiling.
2. Static tool outputs for NPU, CPU, NUMA, and SSD readiness.
