关键性能事实：
- TTFT median：prefix-cache on `8.277s`，off `12.917s`。
- TPOT median：on `137.4ms`，off `270.7ms`。
- E2EL/client-wall median：on `16.405s`，off `30.025s`。

- on/prefill RSS avg `5109.184 MB`、PSS avg `4377.083 MB`。

- `shape_row_count=160`、`op_type_row_count=33`。

- MatMulV2/MatMulV3 有 shape-derived FLOPs denominator 和 input/output tensor footprint bytes。
- 多数 custom kernel 有 tensor footprint bytes，但 FLOPs denominator 仍缺。

边界：P1.28 是 raw counter readout，不是 benchmark。P1.29 是 vLLM OpenAI streaming client 指标，不是 MindIE native run；它可以作为 scoped performance facts，但仍不证明 compute-bound、memory-bound、scheduler-bound、HBM bottleneck、prefix-cache memory benefit 或通用优化结论。P1.30 是 whole-device HBM 与 process-group RSS/PSS readout，不是 per-request KV object bytes 或 HBM traffic。tensor footprint bytes 不是 HBM traffic，msprof raw duration 不是用户侧 latency。
