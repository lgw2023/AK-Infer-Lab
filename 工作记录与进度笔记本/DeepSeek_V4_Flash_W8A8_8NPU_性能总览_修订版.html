核心结论
总体效应：在冻结的 W8A8 / TP8+EP 配置下，MTP 在 24 个 paired batch 中一致降低 E2EL 与 TPOT 并提高 batch output throughput，三项相对变化均值分别为 −40.9%、−47.0%、+71.0%。

证据强度：P6 已形成 5/5 长上下文门、18/18 用户性能矩阵、108/108 matched request 与 4,896 accepted token 活动；结论限于固定 Off→On、每 cell 3 次重复的描述性机制效应。
